# API Structure (Next.js API Routes)

## /api/prayertimes
- Proxy: AlAdhan API  
- Params: lat, lng, method  
- Returns: JSON of daily prayer times  
**V1**

## /api/hijri
- Proxy: AlAdhan API  
- Returns Hijri date and Ramadan start/end  
**V1**

## /api/quran
- Proxy: AlQuran Cloud API  
- Returns ayah of the day  
**V1**

## /api/hadith
- Proxy: Sunnah.com API  
- Returns hadith of the day  
**V1**

## /api/places
- Proxy: Google Places API  
- For halal/mosques queries  
**Later**
